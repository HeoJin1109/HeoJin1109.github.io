e:Tde5,<h1 id="tailwind-css-활용법-가이드">Tailwind CSS 활용법 가이드</h1>
<p>Tailwind CSS는 <strong>Utility-First</strong> 철학을 바탕으로 빠르게 UI를 구축할 수 있게 도와줍니다. 이 포스트에서는 실무에서 유용하게 사용할 수 있는 기법들을 정리합니다.</p>
<hr>
<h2 id="1-주요-특징-및-장점">1. 주요 특징 및 장점</h2>
<ul>
<li><strong>생산성 향상</strong>: 클래스명 고민 없이 즉시 스타일링이 가능합니다.</li>
<li><strong>일관성 유지</strong>: 미리 정의된 디자인 시스템(Spacing, Colors 등)을 따릅니다.</li>
<li><strong>최적화</strong>: PurgeCSS(v3 이상은 내장 엔진)를 통해 사용하지 않는 CSS를 자동으로 제거합니다.</li>
</ul>
<blockquote>
<p>"Tailwind CSS는 단순한 프레임워크가 아니라 스타일링을 대하는 새로운 방식입니다." — <em>디자인 전문가</em></p>
</blockquote>
<hr>
<h2 id="2-코드-예제-react--typescript">2. 코드 예제 (React + TypeScript)</h2>
<h3 id="컴포넌트-스타일링">컴포넌트 스타일링</h3>
<p>반응형 디자인과 Hover 효과를 적용한 간단한 버튼 예제입니다.</p>
<pre><code class="language-tsx">import React from 'react';

interface ButtonProps {
  label: string;
  primary?: boolean;
}

export const CustomButton = ({ label, primary }: ButtonProps) => {
  return (
    &#x3C;button className={`
      px-6 py-2 rounded-lg font-medium transition-colors duration-200
      ${primary 
        ? 'bg-blue-600 text-white hover:bg-blue-700 active:bg-blue-800' 
        : 'bg-gray-100 text-gray-800 hover:bg-gray-200'}
    `}>
      {label}
    &#x3C;/button>
  );
};
</code></pre>
<h3 id="테마-커스텀-tailwindconfigts">테마 커스텀 (tailwind.config.ts)</h3>
<p>프로젝트만의 색상이나 폰트를 추가하는 방법입니다.</p>
<pre><code class="language-javascript">/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    extend: {
      colors: {
        'brand-primary': '#3b82f6',
        'brand-dark': '#1e293b',
      },
      spacing: {
        '128': '32rem',
      }
    },
  },
  plugins: [],
}
</code></pre>

<h2 id="3-기능-비교표">3. 기능 비교표</h2>
<p>| 기능 | CSS-in-JS (Emotion/Styled) | Tailwind CSS | 일반 CSS (Module) |
| :--- | :---: | :---: | :---: |
| 학습 곡선 | 중간 | 낮음 (클래스 숙지 필요) | 낮음 |
| 런타임 비용 | 있음 | 없음 | 없음 |
| 커스텀 자유도 | 높음 | 중간 (설정 필요) | 매우 높음 |</p>

<h2 id="5-유용한-팁">5. 유용한 팁</h2>
<ol>

<li><strong>지시어 사용</strong>: <code>CSS</code> 파일 내에서 <code>@apply</code>를 사용해 반복되는 스타일을 묶을 수 있습니다.</li>
</ol>
